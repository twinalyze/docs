#### How to Send an Ad Event


1. **Ad Request**  
   When an ad fetch is initiated.

<CodeGroup>

```javascript Kotlin
Analytics.getInstance()
    .setAdsRequestEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.Request,
        context,
        adId,
        uniqId
    )
```

```javascript Java
Analytics.getInstance()
    .setAdsRequestEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.Request,
        context,
        adId,
        uniqId
    );
```

</CodeGroup>

2. **Ad Load**  
   When the ad successfully loads.
   
<CodeGroup>

```javascript Kotlin
Analytics.getInstance()
    .setAdsLoadEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.Loaded,
        context,
        adId,
        uniqId
    )
```

```javascript Java
Analytics.getInstance()
    .setAdsLoadEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.Loaded,
        context,
        adId,
        uniqId
    );
```

</CodeGroup>

3. **Ad Failed**  
   Captures load/show errors with error codes and messages.

<CodeGroup>

```javascript Kotlin
Analytics.getInstance()
    .setAdsFailedEvent(
            AdsCompany.Admob,
            AdsType.Interstitial, 
            AdsAction.Failed,
            context,
            adId,
            uniqId,
            null,
            loadAdError == null ? Constant.not_found : String.valueOf(loadAdError.getCode()),
            loadAdError.getMessage() == null ? Constant.not_found : loadAdError.getMessage()
        )
```

```javascript Java
Analytics.getInstance()
    .setAdsFailedEvent(
            AdsCompany.Admob,
            AdsType.Interstitial, 
            AdsAction.Failed,
            context,
            adId,
            uniqId,
            null,
            loadAdError == null ? Constant.not_found : String.valueOf(loadAdError.getCode()),
            loadAdError.getMessage() == null ? Constant.not_found : loadAdError.getMessage()
        );
```

</CodeGroup>
   

4. **Ad Failed Other**  
   Captures load/show errors with error codes and messages. As an example: onAdFailedToShowFullScreenContent.

<CodeGroup>

```javascript Kotlin
Analytics.getInstance()
    .setAdsFailedEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.FailedOther,
        context,adManagerInterstitialAd.getAdUnitId() == null ?
        Constant.not_found : adManagerInterstitialAd.getAdUnitId(),
        uniqId, null,
        String.valueOf(adError.getCode()),
        adError.getMessage()
    )
```

```javascript Java
Analytics.getInstance()
    .setAdsFailedEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        AdsAction.FailedOther,
        context,adManagerInterstitialAd.getAdUnitId() == null ?
        Constant.not_found : adManagerInterstitialAd.getAdUnitId(),
        uniqId, null,
        String.valueOf(adError.getCode()),
        adError.getMessage()
    );
```

</CodeGroup>

7. **Ad Paid Event**  
   Records revenue data only uses AdMob (micros, currency, precision).

<CodeGroup>

```javascript Kotlin
Analytics.getInstance()
    .setAdsPaidEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        context,
        adManagerInterstitialAd.getAdUnitId(),
        uniqId,
        null,
        adValue.getCurrencyCode(),
        String.valueOf(adValue.getPrecisionType()),
        String.valueOf(adValue.getValueMicros())
    )
```

```javascript Java
Analytics.getInstance()
    .setAdsPaidEvent(
        AdsCompany.Admob,

```

</CodeGroup>
